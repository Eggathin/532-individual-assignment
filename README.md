# Building Permits Dashboard

## About

A simple R application for displaying building permits statistics, based on the Group 25 Building Permits project.

[Posit Connect Cloud Deployment]<https://019ce59e-83a7-0829-8b3f-a0b128edefd8.share.connect.posit.cloud/>

## Installation Instructions

Install the required R packages by entering the following R console command:

```r
install.packages(c("shiny", "bslib", "dplyr", "ggplot2", "lubridate", "readr"))
```

## Running the App

Clone the repository to local and enter the following R console command:

```r
shiny::runApp()
```
